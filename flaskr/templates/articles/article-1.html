CSS Layouts

Flow Layout

The Flow Layout, also called the Normal Flow, is the original CSS layout, from the days when the Web was just a collection of text documents with links. In other words, it is the layout that mostly resembles a printed book.

The default position in Normal Flow is static. With position static none of the following attributes have effect: top, righ, bottom, left, z-index. That makes sense, as we are simply following the flow, and blocks of content get positioned one after the other, and never overlap.

To remove an element from the flow you need to position it: relative, absolute, fixed, sticky.

Position Removed from flow? New stacking context?
Relative No Yes
Absolute Yes Yes
Fixed Yes Yes
